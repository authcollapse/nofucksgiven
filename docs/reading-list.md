# Reading List

## Books and Courses

- Boneh and Shoup, *A Graduate Course in Applied Cryptography*.
- Katz and Lindell, *Introduction to Modern Cryptography*.
- Cryptopals Crypto Challenges.

## Standards and Specifications

- NIST FIPS 197: AES.
- NIST SP 800-38D: GCM.
- RFC 8439: ChaCha20-Poly1305.
- RFC 5869: HKDF.
- RFC 8446: TLS 1.3.
- RFC 9180: HPKE.
- NIST post-quantum standards for ML-KEM, ML-DSA, and SLH-DSA.

## Test and Benchmark Resources

- Google Wycheproof.
- SUPERCOP and eBACS.
- Official known-answer test vectors from standards bodies and algorithm authors.
